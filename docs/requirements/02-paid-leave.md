# 有給休暇申請機能 — 要求仕様

Q&A 結果をもとに整理。

- `docs/working/requirements/03-paid-leave-request.md` — Q&A 作業ドキュメント

## ロール（既存と同じ）

| ロール | 有給に関する権限 |
| --- | --- |
| 一般社員 | 自分の有給申請・取り下げ・残日数確認 |
| 上長 | 自部署メンバーの有給申請の承認・却下。自分の有給は自己承認 |
| 管理者 | 全社員の有給状況を確認。管理者自身の申請は所属部署の上長が承認（上長兼任なら自己承認） |

## 有給休暇申請

| ID | ストーリー | ロール |
| --- | --- | --- |
| LEAVE-01 | 有給休暇を申請できる（対象期間・種別・理由を入力） | 全ロール |
| LEAVE-02 | 自分が出した有給申請の状態（申請中/承認/却下/取り下げ）を確認できる | 全ロール |
| LEAVE-03 | 承認前の有給申請を取り下げることができる | 全ロール |
| LEAVE-04 | 自部署メンバーの有給申請を一覧で確認できる | 上長 |
| LEAVE-05 | 自部署メンバーの有給申請を承認または却下できる | 上長 |
| LEAVE-06 | 承認された有給は勤怠履歴に「有給休暇」（全日/午前半休/午後半休）として表示される | システム |
| LEAVE-07 | 有給残日数を確認できる（当年度付与 + 前年度繰越 − 使用済み） | 全ロール |
| LEAVE-08 | 承認操作と取り下げ操作が競合した場合、後の操作が楽観ロックエラーになる | システム |

## 申請ルール

### 対象期間

- 1申請で開始日〜終了日の範囲指定が可能（連続休暇を1回で申請）
- 過去日への事後申請も可能（例: 体調不良で休んだ日を後から申請）

### 種別（半休対応）

| 種別 | 消化日数 | 備考 |
| --- | --- | --- |
| 全日 | 1.0日/対象日 | 終日休暇 |
| 午前半休 | 0.5日/対象日 | 午前のみ休暇。午後は出勤（打刻が必要） |
| 午後半休 | 0.5日/対象日 | 午前は出勤（打刻が必要）。午後休暇 |

- 半休の場合でも申請期間は1日単位（半休 + 連続日数の組み合わせは各日同じ種別）
- 半休日は打刻が必要（出勤時間は半日分として扱う）

### 有給日と打刻の関係

- 全日有給の日は打刻不要。勤怠履歴に「有給休暇」として表示される
- 半休の日は出勤側の半日分の打刻が必要

### バリデーション

- 有給残日数を超える申請はエラー
- 既に有給が承認されている日への重複申請はエラー
- 申請期間に土日が含まれる場合、土日は有給消化の対象外（平日のみカウント）

## 承認フロー

勤怠修正と同じルール:

| 申請者 | 承認者 |
| --- | --- |
| 一般社員 | 所属部署の上長 |
| 上長 | 自己承認 |
| 管理者（上長でない） | 所属部署の上長 |
| 管理者（上長を兼任） | 自己承認 |

---

## 有給付与ルール

### 付与日数（労基法準拠）

| 勤続年数 | 付与日数 |
| --- | --- |
| 0.5年（6ヶ月） | 10日 |
| 1.5年 | 11日 |
| 2.5年 | 12日 |
| 3.5年 | 14日 |
| 4.5年 | 16日 |
| 5.5年 | 18日 |
| 6.5年以上 | 20日 |

- 付与基準日: 入社日から起算（入社日 + 6ヶ月が最初の付与日、以降1年ごと）
- 付与タイミング: 基準日到達時に自動付与

### 繰越

- 前年度分の未使用有給は翌年度に繰り越す
- 繰越上限: 2年分（前年度未使用分のみ。2年前の分は消滅）
- 消化順序: 繰越分（古い方）から先に消化する

---

## 月次集計への反映

- 月次集計に「有給取得日数」を集計項目として追加する
- 有給取得日は「欠勤日数」から除外する
- 欠勤日数 = 営業日（平日）− 出勤日数 − 有給取得日数

---

## 画面

- サイドバーに独立した「有給休暇」メニュー項目を追加する
- 画面構成:
  - 有給残日数サマリー（当年度付与 + 繰越 − 使用済み = 残日数）
  - 有給申請フォーム
  - 自分の申請一覧（ステータス付き）
  - （上長のみ）部署メンバーの申請一覧 + 承認/却下ボタン

---

## スコープ外

- 特別休暇（慶弔休暇、産休、育休等）
- 有給の計画的付与（会社指定の有給取得日）
- 有給取得率のアラート・レポート
- 祝日カレンダーとの連携（土日のみ非営業日として扱う）
